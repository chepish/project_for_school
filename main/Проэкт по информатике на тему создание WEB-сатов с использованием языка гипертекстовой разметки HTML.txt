Занимаясь данным проектом, я поставил себе такие цели:
•	Закрепить знание HTML
•	Закрепить базовые знание селекторов в CSS

Что использовал при работе над проектом?
Во время работы с проектом я использовал такие программы:
•	Visual Studio Code – текстовый редактор, для редактирования кода
•	Figma – графический редактор, для просмотра макета
•	Google Chrome – браузер, для просмотра сайта
И такие инструменты: 
•	HTML, CSS, JS
•	SASS – препроцессор CSS, для ускорения написания стилей 
•	Gulp.js – для сборки проекта
•	Node.js – нужен для работы gulp
